## 实用命令

``` shell
cat /etc/sevices # 查看所有知名的应用层协议

cat /proc/net/arp # 查看 ARP 高速缓存表
ip neigh show     # 查看 ARP 高速缓存表，neigh：neighbourhood

cat /etc/resolv.conf # 查看存放 DNS 服务器的 IP 地址
nslookup www.baidu.com # 查看域名的 IP 地址
host www.baidu.com
dig www.baidu.com +short
```
